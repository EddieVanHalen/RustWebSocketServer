# EchoNet - Real-Time Messaging Server

Welcome to **EchoNet**, a cutting-edge real-time messaging server built in **Rust**. Designed to offer seamless and secure communication for modern applications, EchoNet ensures speed, reliability, and ease of use.

## Features

- **Real-time Messaging**: Fast and efficient message delivery with low latency.
- **Scalability**: Built with scalability in mind, handling large numbers of concurrent connections.
- **Security**: Leverages the latest encryption protocols to ensure secure communication.
- **Rust Performance**: Written in Rust for maximum performance and memory safety.

## 🛠️ Installation

### Requirements

- [Rust](https://www.rust-lang.org/): Make sure Rust is installed on your system. If not, you can install it by following the instructions [here](https://www.rust-lang.org/tools/install).
  
### Clone the Repository

```bash
git clone https://github.com/yourusername/EchoNet.git
cd EchoNet
```

### Install Dependencies

Once the repository is cloned, run the following command to install the dependencies:

```bash
cargo build --release
```

### Run the Server

Start the EchoNet server by running:

```bash
cargo run
```

By default, the server will listen on port 8080. You can customize the port and other settings by modifying the configuration file.

## 📝 Usage

Once the server is up and running, EchoNet allows you to:

- **Send messages** to other connected clients.
- **Join groups** for chatrooms or private discussions.
- **Monitor message delivery status** in real-time.

### API Endpoints

- `POST /send`: Send a message to a connected user or group.
- `GET /status`: Check the status of a message.
- `GET /history`: Retrieve message history.

More endpoints will be added as development progresses.
